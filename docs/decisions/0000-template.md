# ADR 0000: Use lightweight ADRs

## Status

Accepted

## Context

Teams and AI agents need a single place to read **why** something is true.

## Decision

Use numbered ADRs in `docs/decisions/` with short filenames: `NNNN-title-kebab-case.md`.

## Consequences

- Easier onboarding and fewer repeated debates.
- Requires discipline: update ADRs when decisions change.
